# SolidCompressor
Compressor library for large files. Files > 200 MB will do justice to the project. 
Dependencies : libcrypto : Hash (SHA-2)
               libfuzzy  : SSDEEP
               libzd     : zdelta
               libz      : gzip 
               libpthread: Posix threads
               libbz2    : bzip2
Future support for 7z in the pipe.
